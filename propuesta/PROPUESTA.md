## Descripción de cada imágen

* **(home)**: Pantalla de inicio que se verá al entrar a `localhost:3000`
* **(login)**: Pantalla para iniciar sesión en tu cuenta
* **(sing_up)**: Pantalla para crear una cuenta
* **(interfaz_juego)**: Pantalla principal donde se mostrarán las preguntas y opciones de respuesta
* **(Dashboard)**: Pantalla para ver el rendimiento personal del usuario y puntuaciones
* **(Admin_Panel)**: Pantalla para gestionar las preguntas que se van generando con la API de Google Gemini.
